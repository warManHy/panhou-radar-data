# 盘后雷达公开数据

本仓库保存盘后雷达每天生成并校验的 A 股盘后复盘 JSON，供 APP 刷新和按日期查询。数据由用户本机 Mac 在交易日自动同步。

## 固定地址

- `data/index.json`：日期索引、最新交易日及各日文件摘要
- `data/latest.json`：最新一个已发布交易日的完整快照
- `data/reviews/YYYY-MM-DD.json`：指定交易日的完整快照

所有文件当前使用 `schemaVersion: 1`。板块、股票与连板数量按当天真实结果动态变化，不是固定数量。`generatedAt` 等时间字段可能使用 UTC 的 `Z` 后缀，也可能使用 `+08:00`，读取端应按 ISO 8601 解析并以北京时间展示。

主要行情来源包括东方财富与腾讯行情；同步前会执行结构、完整性和连板数据校验。公开数据仅供市场复盘和技术研究，不构成任何投资建议。
